# Soni Family Foundation Website

Static GitHub Pages website.

## Publish
Upload the contents of this folder to the repository branch currently used by GitHub Pages. Keep `CNAME` in the root so the custom domain remains connected.

## Edit
- Page content: the HTML files in the root
- Global design: `css/style.css`
- Navigation and scroll effects: `js/main.js`
- Photos and logo: `assets/`
